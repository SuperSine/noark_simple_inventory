Requirements

This script is written in PHP and utilizes the power of Laravel framework. It requires:


  • PHP 5.3+
  • Apache webserver
  • MySQL 5+
  • PDO Extension for PHP (MySQL)
  • MCrypt Extension for PHP
  • Fileinfo Extension for Image uploads
